# Meu Portfólio Pessoal

Este é um site de portfólio responsivo desenvolvido como parte de um projeto prático para demonstrar habilidades em desenvolvimento web front-end. O projeto utiliza HTML5 semântico, estilização moderna com CSS (Flexbox e Grid), e interatividade via JavaScript.

## 🚀 Funcionalidades
* **Navegação estruturada:** Três páginas totalmente funcionais (`Início`, `Sobre Mim` e `Projetos`).
* **Alternador de Tema (Dark Mode):** Recurso em JavaScript que altera o tema visual do site e persiste a escolha do usuário utilizando `localStorage`.
* **Layout Responsivo:** Construído com CSS Grid e Flexbox para se adaptar a diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Tags semânticas validadas pelo Markup Validation Service do W3C)
* **CSS3** (Variáveis nativas, Flexbox e Grid Layout)
* **JavaScript (ES6+)** (Manipulação do DOM e API de Armazenamento Local)

## 📌 Conteúdo do Site
* **Interesses Profissionais:** Listagem detalhada dos focos de estudo na área de tecnologia.
* **Projeto 1:** Seção dedicada à descrição técnica de um artefato de software desenvolvido previamente.
* **Links de Estudo:** Uma curadoria de referências importantes para a formação profissional.
